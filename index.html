I love to always have some kind of project going on. Whether that's for class or just for a random challenge, I'm always working on something. I really enjoy the challenge aspect of it. Here are some projects that I have worked on...

Atari Centipede

I was tasked with making a MIPS-version of the famous Atari game, Centipede. I had to follow all of the conventional rules and game mechanics, but was free to add other small tidbits to make the game more enjoyable. I ended up adding native MIPS sounds using syscall 31 with various parameters. This was for CS 0447 during Fall of Sophomore year.
Boggle in Java

I made a recursive program in Java that returns all 3+ letter words from a 10x10 board of random letters. Words are cross-checked against a provided dictionary. This was for CS 0445 during Fall of Freshman year.
BookBot

You can look email multiple files to a single email to a single person, after which the entire directory will be wiped clean. I made this mainly for my kindle. This was a personal project, developed in Winter of 2021.
EmailClient

Use a simple CSV file to send personalized messages to countless people, complete with HTML signatures. This was a personal project I developed for my father in Spring of 2021.
